## 5. Architecture Constraints

### 5.1 Dependency Rules
- Domain layer MUST have zero dependencies on outer layers
- Application layer depends only on Domain interfaces
- Infrastructure implements Domain interfaces
- Interface layer depends on Application layer

### 5.2 Layer Isolation Requirements
- All cross-layer communication via interfaces
- No direct database access from domain
- DTOs for API responses only

### 5.3 Event Publishing Guidelines
- Domain events published synchronously within aggregate
- Application layer handles event dispatch
- Eventual consistency for cross-context communication
